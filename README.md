## Gestão de Empresas - API

#### Sistema de gestão de empresas e funcionários, podendo adicionar empresas e realizar o controle de  números de funcionários que atuam nela atualmente.

#### Tecnolgies

```aidl
 Java 8
 Spring Boot
 Hibernate
 JPA
 Maven
 H2 Database
 Lombok
```

### Clone project
```
git clone https://github.com/Deborah-Proenca/gestao-empresarial.git
cd gestao-empresarial
```

### Run project

```
mvn spring-boot:run
```
